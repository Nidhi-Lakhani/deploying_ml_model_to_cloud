# Deploying an ML model to Cloud 

This is a repository created by Nidhi Lakhani as a part of Udacity's ML DevOps Nanodegree - Deploying Scalable ML Pipeline in Production. This course teaches us to deploy an ML model to cloud using FastAPI and a cloud deployment service such as Render.


## Summary

This project covers - data cleaning, exploratory data analysis, model development, unit testing, creating model card, setting up CI/CD pipelines, connecting with FastAPI and finally, deploying the model to Render.


## Relevant Links

The GitHub repo - https://github.com/Nidhi-Lakhani/deploying_ml_model_to_cloud

Render app - https://ml-to-cloud.onrender.com/

It is set to auto deploy once CI passes. All the relevant screenshots can be found under the screenshots folder.
